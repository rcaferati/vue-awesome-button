# Changelog

## 1.0.1

- added fixed-size width/height transitions for `AwesomeButton`
- added measured auto-width transitions for `AwesomeButton`
- added `animateSize` as an opt-out prop for base, progress, and social buttons
- added Storybook coverage for animated fixed-size and auto-width changes

## 1.0.0

- initial stable release of `@rcaferati/vue-awesome-button`
- shipped `AwesomeButton`, `AwesomeButtonProgress`, and `AwesomeButtonSocial`
- added press/release interaction hardening, auto-width snapping, ripple support, and string-only text transitions
- added guarded progress lifecycle with `showProgressBar` and `progressLoadingTime`
- added built-in social sharer flows for facebook, twitter, pinterest, linkedin, reddit, whatsapp, messenger, mail, and instagram
- bundled 10 themes and full Storybook sections for Components, Icons, and Themes
